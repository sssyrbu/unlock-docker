# Unlock docker
Unlocks docker hub in Russia, Cuba, Iran, North Korea, Sudan, and Syria.

Works on Linux machines with Systemd, Upstart and Runit as initialization systems.

### How to Use
> Make the script executable
```bash
  chmod +x unlock.sh
 ```
> Run the script
```bash
./unlock.sh
```
